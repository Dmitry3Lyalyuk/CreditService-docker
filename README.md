Credit Service — Система управления займами
Это полнофункциональное (Fullstack) веб-приложение для управления займами, которые создают пользователи. Проект построен на современной микросервисной архитектуре с использованием Docker для контейнеризации всех компонентов.
Backend:
Платформа: .NET 8 (Web API).
Архитектура: Многослойная (Domain, Application, Infrastructure, Web).
ORM: Entity Framework Core с провайдером Npgsql.
Db: PostgreSQL.
Frontend:
Фреймворк: Vue 3
